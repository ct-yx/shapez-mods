安装只需要三步。

整合包内的每个 JS 文件均可以单独复制到游戏的 Mod 目录中。

01 · 下载

下载 ZIP 整合包，解压后进入其中的 mods/ 目录。

02 · 复制

将需要的 .js 文件复制到 shapez.io 的 Mod 目录。macOS 常用路径为 ~/Library/Preferences/shapez.io/mods/。

03 · 启用

重启游戏后在 Mod 管理界面启用。需要调参数时，打开「设置 → 游戏模组（MODS）」。
