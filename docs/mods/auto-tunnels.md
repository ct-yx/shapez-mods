# Auto Tunnels

> 方向锁定铺传送带时，自动在合适的位置用地下传送带跨越既有物流。

- **分类：**建造
- **版本：**1.0.4
- **文件：**[`auto-tunnels@1.0.4.js`](../../mods/auto-tunnels%401.0.4.js)

## 用途

Auto Tunnels 扩展的是传送带方向锁定铺设逻辑。当路线遇到现有传送带或无法直接接收的格子时，它会检查可用距离并尝试放置一对地下传送带，而不是单纯中断铺设。

## 核心功能

- 仅接管普通传送带的方向锁定批量铺设；其他建筑保留原版拖动建造逻辑。
- 依据已解锁的地下传送带等级选择 5 格或 9 格范围。
- 在具备 Smart Tunnel 奖励时，可处理转向场景并使用 smart 变形体。
- 遇到可直接接收物品的目标、同向平衡器或已存在匹配隧道时，会保留常规连接。

## 使用方法

1. 选择普通传送带并使用方向锁定铺设。
2. 穿越既有物流时留出符合当前地下传送带距离的出口位置。
3. 确认地下传送带与 Smart Tunnel 奖励已在当前存档中解锁。

## 兼容性与注意事项

- 该文件来自 Auto Tunnels Remake 的本地版本；功能是否可用取决于当前游戏的建造器 API。
