# Belt Speed Control

> 让传送带、地下传送带、原版二路平衡器和读取器使用统一的高吞吐倍率。

- **分类：**物流
- **版本：**1.4.0
- **文件：**[`belt_speed_10x.js`](../../mods/belt_speed_10x.js)

## 用途

用于快速验证大型物流网络。它保留游戏原有的等级与升级关系，只是在运行期按配置放大吞吐。默认倍率为 10×，也可以随时回到 1× 原版速度。

## 核心功能

- 每级传送带速度可设置为 1–50×，关闭开关或设为 1× 即恢复原版行为。
- 一级与二级地下传送带距离可分别设置为原版的 1–10×：最高 50 格与 90 格。
- 原版二路平衡器的处理速度同步增加，避免它成为高速带网络中的瓶颈。
- 传送带读取器改为即时透传，并解除原版读取覆盖层的 47.8 items/s 显示上限。

## 使用方法

1. 启用 Structured Mod Settings UI 后，进入「游戏模组（MODS）→ 传送带速度控制」。
2. 先用 5× 或 10× 验证布局，再逐步提高倍率观察吞吐和逻辑负载。
3. 改变地下传送带距离后，Mod 会刷新既有隧道连接缓存。
