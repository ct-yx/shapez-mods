Chainable Extractors

拖动铺设链式采矿机时，连续放置并自动调整链路朝向。

版本 1.0.0文件 extractor_chain.jsshapez.io 1.x

用途

这个轻量 Mod 专注于链式采矿机的批量摆放。选中 miner 的 chainable 变形体后拖动鼠标，Mod 会沿拖动路径逐格放置，并根据水平方向和垂直方向更新每一段朝向。

核心功能

  • 仅在当前建筑是 miner 且变形体为 chainable 时接管鼠标移动。
  • 沿拖动轨迹逐格连续放置，适合批量覆盖资源块。
  • 使用反向放置修饰键时，按相反方向计算旋转。
  • 地图总览激活、方向锁定或镜头正在移动时回退到原版处理。

使用方法

  • 选择采矿机的链式变形体。
  • 按住鼠标并横向、纵向或斜向拖动，连续铺设链条。
  • 需要反向链路时配合游戏原有的反向放置修饰键。

兼容性与注意事项

它直接修改建造器的鼠标移动处理逻辑；若其他 Mod 也重写同一方法，应优先确认组合行为。

获取方式

下载整合包后，只需将本 Mod 对应的 JS 文件复制到游戏 Mod 目录。
